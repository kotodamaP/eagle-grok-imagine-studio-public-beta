# Third-Party Tools And Services

This repository ships the Eagle Grok Imagine Studio plugin source and public setup templates only. It does not bundle the following third-party tools, accounts, models, or runtime assets.

## External tools and services

- Eagle and Eagle Plugin API: users must install Eagle separately and follow Eagle's terms and developer documentation.
- Grok / xAI / Grok CLI / Grok Build: users must configure their own Grok environment and follow xAI/Grok terms, quota, and account rules.
- FFmpeg / FFprobe: users must install FFmpeg separately and follow FFmpeg's license terms for their chosen build.
- Upscayl: optional local image upscale helper; users must install it separately and follow Upscayl's license terms.
- Aratako/Irodori-TTS: optional narration helper; users must install it separately and follow the upstream repository, model card, and license terms.
- Eagle browser extensions: optional separate tooling for collecting web assets before using this plugin.

## Not bundled

- No Grok login state, sessions, cookies, tokens, or direct API credentials.
- No Eagle app binaries or browser extensions.
- No FFmpeg, FFprobe, or Upscayl binaries.
- No Irodori-TTS source tree, model weights, datasets, reference audio, or generated audio.
- No generated images/videos, local logs, or private Eagle libraries.
